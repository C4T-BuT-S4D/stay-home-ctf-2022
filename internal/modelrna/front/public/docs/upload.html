Follow this guide to upload your model

  1. You need to register first.
  2. You need to train your model and convert it to the ONNX format.
  3. First, download the sample dataset.
    Second, train and convert your model using this sample script (will use the catboost library).
    (You can train and use any other ONNX model as long as it use the same input format and work with the same features)
  4. Upload your model using UI or API (described below)

To upload your model using the API follow these steps:

  1. Download this sample python script.
  2. Find your API_TOKEN and VACCINE_KEY on home page.
  3. Fill the "PATH_TO_MODEL"(path to your ONNX model), "API_TOKEN", "VACCINE_KEY", "HOST"(domain) variables
